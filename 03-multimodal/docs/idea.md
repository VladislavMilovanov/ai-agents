Идея: создать персонального финансового советника в формате Telegram-бота на базе LLM/VLM.

Бот помогает вести учёт личных доходов и расходов прямо в чате:
- принимает текстовые сообщения о тратах и поступлениях;
- принимает фотографии чеков;
- извлекает данные в структурированном виде;
- сохраняет транзакции в in-memory таблице;
- по запросу формирует отчёт о балансе за период с разбивкой по категориям.

MVP-формат транзакции:
- дата и время;
- направление (доход/расход);
- сумма;
- тип расхода (повседневный/периодический/разовый);
- категория;
- описание.

Извлечение из текста выполняется LLM со structured output.
Извлечение из фото чеков выполняется VLM напрямую по изображению.
Провайдер и модели переключаются через конфиг: OpenRouter или локальный Ollama.

Фокус MVP: простой end-to-end сценарий без оверинжиниринга, строго по KISS и YAGNI.
