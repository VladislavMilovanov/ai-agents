Идея проекта — разработка LLM-ассистента в виде Telegram-бота.

Бот будет вести диалог с пользователем, отвечая на вопросы и взаимодействуя на основе заранее заданной роли из системного промпта LLM. Основная цель — обеспечить удобную коммуникацию и поддержку пользователей через Telegram с учетом роли, определённой разработчиком для LLM-ассистента.
